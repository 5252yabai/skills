# Persona Builder

사용자의 실제 발화·글을 근거로 **대리 판단/응답용 1인칭 페르소나 문서**를 만든다.
에이전트가 이 문서를 로드하면 "특정 상황에서 사용자라면 어떻게 판단·반응·응답할지"를 흉내낼 수 있다.

**이 스킬은 빌드 전용이다.** 페르소나 문서를 생성·갱신만 한다. 적용(대리 판단)은 만들어진 문서를 컨텍스트로 로드해 다른 에이전트가 수행한다.

## 고정 설정

- **위키 경로**: `/Users/leejaeseok/texts_wiki`
- **산출물**: `2.Wiki/analyses/{사용자명}-페르소나.md` (기본 `이재석-페르소나.md`)
- **Slack 신원**: 이메일 `jslee01@miridih.com` → 빌드 시 `slack_search_users`로 user id/handle 해석
- **데이터 범위**: Slack 전체(공개+비공개+DM) + `1.Raw/` 전부. **민감 채널·DM·사적 글 전부 포함** (사용자 동의됨)
- **추출 4차원**: 반응 패턴 / 가치 판단 기준 / 커뮤니케이션 스타일 / 의사결정 방식
- **상황 공간**: 업무 의사결정 · 코드리뷰/기술 판단 · 대인 응답/조율 · 문서 작성
- **충실도**: 있는 그대로 (편향·조급함·감정·반복 실수 포함, 미화 금지)

상세 추출 방법·Slack 수집 요령·병합 규칙은 [REFERENCE.md](REFERENCE.md) 참조.

## 워크플로우

### 0. 세팅 & 증분 판단

- [ ] `2.Wiki/analyses/{사용자명}-페르소나.md` 존재 확인.
- [ ] 있으면 하단 `## 갱신 상태`에서 **마지막 실행일·처리 범위**를 읽는다 → 이후 데이터만 수집(증분).
- [ ] 없거나 사용자가 "전체 재빌드" 요청 시 → 전량 수집(최초 빌드).
- [ ] `slack_search_users`로 이메일→Slack handle 해석.

### 1. 수집 (증분 창)

- [ ] **Slack**: `slack_search_public_and_private`로 `from:<handle>` + 날짜 필터로 사용자 발화 수집. 각 발화마다 **맥락(무엇에 대한 반응인지)**을 `slack_read_thread`/`slack_read_channel`로 함께 확보 → (맥락 → 내 응답) 쌍.
- [ ] **위키**: `1.Raw/` 중 증분 창 내 파일(폴더 `YYYY-MM`/mtime/`git log`). `ai_session/`은 사용자 발화 턴만 추출.
- [ ] 전량이 크면 월 단위 배치로 나눠 처리하고, **샘플링·누락한 범위를 반드시 로그**한다(조용한 절단 금지).

### 2. 추출 (4차원 × 상황 공간)

- [ ] 근거에서 규칙을 뽑는다. **규칙마다 실제 발화/문서 인용 + 출처 링크**(Slack permalink 또는 `[[1.Raw/문서]]`) 필수.
- [ ] 있는 그대로: 편향·조급함·감정 반응·반복 실수도 규칙으로 남긴다.
- [ ] 각 규칙에 상황 공간 태그를 붙인다.

### 3. 합성 (1인칭 문서)

- [ ] `assets/persona-template.md` 구조로 작성/병합. **1인칭 목소리**("나는…").
- [ ] 증분이면 기존 규칙에 새 근거를 병합·강화하고, 모순·변화는 진화 기록으로 남긴다(REFERENCE 병합 규칙).

### 4. 캘리브레이션 세트

- [ ] 상황 공간별 `시나리오 → 예측 반응` 쌍을 표로 생성(검증용). 사용자가 나중에 채점해 정확도를 높인다.

### 5. 저장 & 로그

- [ ] `2.Wiki/analyses/{사용자명}-페르소나.md` 저장. `## 갱신 상태`에 실행일·처리 범위·커버리지 메모 갱신.
- [ ] `2.Wiki/index.md`에 항목 추가/갱신(analyses, 가나다순).
- [ ] `2.Wiki/log/YYYY-MM.md`에 기록: `- 생성/갱신: 설명 ([[analyses/…]], 출처)`.

## 위키 규칙 (필수 준수)

- 위키 페이지 본문에 **H1(`#`) 금지** — `##`부터. 파일명이 곧 제목.
- **한국어**로 작성.
- 링크는 `[[wikilink]]`. 원본은 `[[1.Raw/문서명]]`.
- Category/Topic/Tag 금지 — 조직은 wikilink로만.
- `1.Raw/`는 **읽기 전용** — 절대 수정하지 않는다.
